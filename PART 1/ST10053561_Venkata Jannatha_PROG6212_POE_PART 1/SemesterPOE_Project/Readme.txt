##Student Academic Time Management System
This project is designed to facilitate academic time management for students, providing a clear overview of the time required for each module. The system comprises several key components:

- Opening Menu: This serves as the initial interface for the project.
- Semester Modules Capture: This feature allows for the input of module information.
- Semester Dates: This function records the commencement date of the semester and the duration of each course.
- Display Academics: This displays the remaining hours required for each course.

To execute the program, please adhere to the following instructions:

1. Extract the contents of “ST10053561_Venkata Jannatha_PROG6212_POE_PART1”.
2. Open the “SemesterPOE_Project.sln” file using Visual Studio software.
3. Click on the ‘Start’ button (represented by a green arrow) located at the top of the IDE window to launch the application.
